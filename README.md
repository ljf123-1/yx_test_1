# 小宇播客 - 高保真原型

这是一个类似小宇宙的播客应用高保真原型，使用HTML、CSS和JavaScript构建。原型设计遵循iOS设计规范，提供了流畅的用户体验和精美的界面设计。

## 功能特点

- **精美的UI设计**：符合Apple设计语言，视觉效果精美
- **响应式布局**：适配iPhone 15 Pro尺寸，模拟真实手机界面
- **交互式原型**：可点击操作的界面元素，模拟真实应用体验
- **多页面导航**：包含主页、发现、播客详情、播放器、个人主页和设置等页面
- **iOS风格组件**：包含状态栏、底部导航栏等iOS原生组件
- **实时动态效果**：播放器控制、进度条交互、播放速度调整等

## 页面说明

该原型包含以下页面：

1. **首页 (home.html)**：展示个性化推荐和热门播客
2. **发现页 (discover.html)**：分类浏览播客内容
3. **播客详情页 (podcast-detail.html)**：展示播客信息和剧集列表
4. **播放器页 (player.html)**：提供媒体播放控制和信息展示
5. **个人主页 (profile.html)**：用户个人信息和订阅内容管理
6. **设置页 (settings.html)**：应用设置选项

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript**：交互逻辑
- **Tailwind CSS**：响应式样式库
- **Font Awesome**：图标库

## 如何使用

1. 直接打开 `index.html` 文件，查看所有页面原型平铺展示
2. 点击任何一个页面内的界面元素，体验交互效果
3. 所有页面都有完整的导航功能，可以在不同页面之间切换
